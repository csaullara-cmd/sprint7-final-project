# sprint7-final-project
##Proyecto ConnectaTel
Este repositorio contiene el analisis del proyecto final de sprint 7 de una empresade telecomunicaciones
**Objetivo**:
1:limpiar y estandarizar datos nulos y sentinesl en los data frames
2:identifacar diferentes segmentaciones de usuarios 
##Dataset utilizados:
/datasets/plans.csv     Tipos de planes de telecomunicacion
/datasets/users_latam.csv  Lista de usuarios 
/datasets/usage.csv        Uso de los planes de los usuarios
##Etapas del analisis:
-Identificacion de datos nulos, inexistentes y sentinels en los datasets
-Limpieza de datos nulos y snetinels
-Revision y estandarizacion de fechas
-Visualizacion de distribuciones y outlers
-Segmantacion de clientes por edad y uso
